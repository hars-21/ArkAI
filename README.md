# ArkAI – Privacy-First AI Shopping Assistant

[![Flutter](https://img.shields.io/badge/Flutter-3.x-blue?logo=flutter)](https://docs.flutter.dev/)
[![Dart](https://img.shields.io/badge/Dart-3.x-0175C2?logo=dart)](https://dart.dev/)
[![License: MIT](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)

ArkAI is a privacy-first shopping assistant that helps users make better purchase decisions while browsing e-commerce platforms. It focuses on smarter buying, sustainability awareness, and a clean user experience.

All core intelligence is designed with a privacy-first approach, keeping user trust and local-first behavior at the center.

---

## What’s New

ArkAI now spans multiple touchpoints in one ecosystem:

- **Mobile App** for a native shopping experience
- **Browser Extension** for direct in-browser assistance
- **Backend Services** to support intelligent workflows

This makes ArkAI more accessible and consistent across how users shop.

---

## Overview

ArkAI is built for people who want quick, meaningful guidance before they buy.

- Helps users compare value and sustainability at a glance
- Supports modern e-commerce browsing flows
- Keeps the product direction focused on simplicity and trust

---

## Privacy

Privacy is a core principle of ArkAI:

- No unnecessary tracking
- No intrusive data collection
- User-first design decisions

---

## Quick Start

```bash
git clone https://github.com/hars-21/ArkAI.git

```

Then run the part of the project you want to work on (`mobile_app`, `extension`, or `backend`).
